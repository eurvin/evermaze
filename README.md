# evermaze
An ever evolving and transforming maze

## Inspiration
This project is inspired by Casey Reas - [10 PRINT](https://www.youtube.com/watch?v=m9joBLOZVEo)

## project outline
1) procedurally generate a maze.
2) then periodically update / change the maze to make it ever evolving.
